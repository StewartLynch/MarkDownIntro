## HORIZONTAL RULE

Place a horizontal rule between these two lines of text.

## EMPHASIS

**This text should be bold**
__So should this text__
*This text should be italic*
_This should be italics too_
**_This text should be bold and italic_**
~~This text should be strikethrough~~


## LISTS

**Unordered List**

* First item
* Second item
* Third item
  * Nested item under third

**Ordered List**

1. First item
2. Second item
3. Third item
   1. Nested item under third

**Task List**

- [x] Completed task
- [ ] Incomplete task
- [ ] Another incomplete task


## BLOCKQUOTES

> This is a blockquote. Use it to highlight an important idea or a quote from someone.
>> This is a nested blockquote inside the one above.
